## xPDO::setDebug

Устанавливает состояние отладки для соединения XPDO.

## Синтаксис

API Docs: <http://api.modxcms.com/xpdo/xPDO.html#setDebug>

```php
void setDebug ([boolean $v = true])
```

## Пример

```php
$xpdo->setDebug(true);
```
