## cache_db_expires

-   **Имя**: Время жизни кэша базы данных
-   **Тип**: Number
-   **По умолчанию**: 0

Время по умолчанию для истечения срока действия кэша базы данных. Если установлено значение «0», кэш никогда не истечет, если запись не будет обновлена.
