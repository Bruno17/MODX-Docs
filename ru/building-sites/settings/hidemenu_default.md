## hidemenu_default

-   **Имя**: Скрыть из меню по умолчанию
-   **Тип**: Да/Нет
-   **По умолчанию**: Нет
-   **Доступно в**: 2.0.7+

Выберите «Да», чтобы по умолчанию все новые ресурсы были скрыты от меню.
