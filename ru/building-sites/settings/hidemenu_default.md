-   **Имя**: Скрыть из Меню по умолчанию
-   **Тип**: Да/Нет
-   **По умолчанию**: Нет
-   **Доступно в**: 2.0.7+

Выберите «Да», чтобы по умолчанию все новые Ресурсы были скрыты из Меню.
