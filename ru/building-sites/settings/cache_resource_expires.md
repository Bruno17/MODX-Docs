## cache_resource_expires

-   **Имя**: Время жизни частичного кэша ресурсов
-   **Тип**: Number
-   **По умолчанию**: 0

Время истечения для параметра Partial Resource Cache. Значение 0 означает, что срок действия кэша никогда не истечет.
