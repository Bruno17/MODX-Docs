## phpthumb_error_textcolor

-   **Имя**: phpThumb Цвет текста ошибки
-   **Тип**: textfield
-   **По умолчанию**: FF0000
-   **Доступно в**: 2.0.8+

Шестнадцатеричное значение без знака #, указывающее цвет текста для вывода ошибки phpThumb.
