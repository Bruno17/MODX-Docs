## manager_favicon_url

-   **Имя**: URL фавиконки панели управления
-   **Тип**: Textfield
-   **По умолчанию**:
-   **Доступно в**: 2.1+

Если установлено, этот URL будет загружен как значок для менеджера MODX. Должен быть относительным URL-адресом manager/ каталога или абсолютным URL-адресом.
