## phpthumb_error_bgcolor

-   **Имя**: phpThumb Цвет фона сообщения об ошибке
-   **Тип**: textfield
-   **По умолчанию**: CCCCFF
-   **Доступно в**: 2.0.8+

Шестнадцатеричное значение без знака #, указывающее цвет фона для вывода ошибки phpThumb.
