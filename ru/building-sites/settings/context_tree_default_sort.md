## context_tree_sort

-   **Имя**: Включить сортировку контекстов в дереве ресурсов
-   **Тип**: Да/Нет
-   **По умолчанию**: Нет
-   **Доступно**: 2.1.2+

Если контекст(ы) в дереве ресурсов должны быть отсортированы по системным параметрам context_tree_sortby и context_tree_sortdir
