## cache_json_expires

-   **Имя**: Время истечения срока действия JSON Cache
-   **Тип**: Number
-   **По умолчанию**: 0

Время истечения для кэшированных данных в формате JSON. Значение 0 означает, что кэш никогда не истекает.
