## allow_forward_across_contexts

-   **Имя**: Разрешить перенаправление через контексты

-   **Тип**: Да/Нет
-   **По умолчанию**: Нет
-   **Доступно в**: 2.1+

При включении вызовы API Symlinks и `modX::sendForward()` запросы могут пересылать к ресурсам в других контекстах.
