## Event: OnCategoryRemove

Happens after a category is removed.

- Service: 2 - Manager Access Events
- Group: modCategory

## Event Parameters

| Name     | Description                            |
| -------- | -------------------------------------- |
| category | A reference to the modCategory object. |
