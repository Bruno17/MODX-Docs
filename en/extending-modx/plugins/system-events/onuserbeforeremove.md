## Event: OnUserBeforeRemove

Fires right before a User is removed.

- Service: 1 - Parser Service Events
- Group: modUser

## Event Parameters

| Name | Description                        |
| ---- | ---------------------------------- |
| user | A reference to the modUser object. |
