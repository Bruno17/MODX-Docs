## Event: OnPluginFormDelete

Fires after a plugin is deleted.

- Service: 1 - Parser Service Events
- Group: Plugins

## Event Parameters

| Name  | Description                          |
| ----- | ------------------------------------ |
| chunk | A reference to the modPlugin object. |
| id    | The ID of the Plugin.                |
