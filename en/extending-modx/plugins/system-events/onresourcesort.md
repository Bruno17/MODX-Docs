## Event: OnResourceSort

Fires just after the Resource tree is sorted.

-   Service: 1 - Parser Service Events
-   Group: Resources

## Event Parameters

| Name   | Description                                         |
| ------ | --------------------------------------------------- |
| `node` | A reference to the modResourceSortProcessor object. |
