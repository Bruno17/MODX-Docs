## Event: OnPluginBeforeSave

Happens right before a plugin is saved.

- Service: 2 - Manager Access Events
- Group: modPlugin

## Event Parameters

| Name   | Description                          |
| ------ | ------------------------------------ |
| plugin | A reference to the modPlugin object. |
