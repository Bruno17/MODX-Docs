## Event: OnResourceGroupSave

Fires after a Resource Group is saved.

- Service: 1 - Parser Service Events
- Group: modResourceGroup

## Event Parameters

| Name  | Description                                 |
| ----- | ------------------------------------------- |
| group | A reference to the modResourceGroup object. |
