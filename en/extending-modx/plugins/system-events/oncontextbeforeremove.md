## Event: OnContextBeforeRemove

Happens right before a context is removed.

- Service: 2 - Manager Access Events
- Group: modContext

## Event Parameters

| Name    | Description                           |
| ------- | ------------------------------------- |
| context | A reference to the modContext object. |
