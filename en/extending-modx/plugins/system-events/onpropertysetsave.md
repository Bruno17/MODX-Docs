## Event: OnPropertySetSave

Fires right after a Property Set is saved.

- Service: 1 - Parser Service Events

## Event Parameters

| Name        | Description                               |
| ----------- | ----------------------------------------- |
| propertySet | A reference to the modPropertySet object. |
