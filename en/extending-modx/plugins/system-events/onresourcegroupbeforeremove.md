## Event: OnResourceGroupBeforeRemove

Fires right before a Resource Group is removed.

- Service: 1 - Parser Service Events

## Event Parameters

| Name  | Description                                 |
| ----- | ------------------------------------------- |
| group | A reference to the modResourceGroup object. |
