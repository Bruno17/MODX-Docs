## Event: OnBeforeUserFormDelete

Fires before a User is deleted in the manager.

- Service: 1 - Parser Service Events
- Group: Users

## Event Parameters

| Name | Description                        |
| ---- | ---------------------------------- |
| user | A reference to the modUser object. |
| id   | The ID of the User.                |
