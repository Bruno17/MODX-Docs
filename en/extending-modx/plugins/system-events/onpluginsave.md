## Event: OnPluginSave

Fires just after the database is altered

- Service: 2 - Manager Access Events
- Group: modPlugin

## Event Parameters

| Name   | Description                          |
| ------ | ------------------------------------ |
| plugin | A reference to the modPlugin object. |
