## Event: OnChunkSave

Happens after a chunk is saved.

- Service: 2 - Manager Access Events
- Group: modChunk

## Event Parameters

| Name  | Description                         |
| ----- | ----------------------------------- |
| chunk | A reference to the modChunk object. |
