## Event: OnMODXInit

Invoked upon initialization of the MODX object.

- Service: 5 - Template Service Events
- Group: System

## Event Parameters

| Name       | Description                                       |
| ---------- | ------------------------------------------------- |
| contextKey | The context\_key of the context being initialized |
| options    | Any options passed to the initialize() function   |
