## Event: OnBeforeTempFormDelete

Fires before a Template is deleted in the manager.

- Service: 1 - Parser Service Events
- Group: Templates

## Event Parameters

| Name     | Description                            |
| -------- | -------------------------------------- |
| template | A reference to the modTemplate object. |
| id       | The ID of the Template.                |
