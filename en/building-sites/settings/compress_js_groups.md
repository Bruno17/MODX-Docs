**Name**: Use Grouping When Compressing JavaScript    
**Type**: Yes/No    
**Default**: No 
**Deleted In**: Revolution 2.5.0+

Group the core MODX manager JavaScript using minify's groupsConfig. Set to 'Yes' if using Suhosin or other limiting factors.
