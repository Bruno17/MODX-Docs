**Name**: Logging Target  
**Type**: String  
**Default**: 'FILE'   
**Available In**: Revolution 2.2.0+

The default logging target where log messages are written. Available options: \'FILE\', \'HTML\', or \'ECHO\'. Default is \'FILE\' if not specified
