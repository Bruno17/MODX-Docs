**Name**: Strict Request Method  
**Type**: Yes/No
**Default**: No   
**Available In**: 2.2.0+

If enabled, requests via the Request ID Parameter will be ignored with FURLs enabled, and those via Request Alias Parameter will be ignored without FURLs enabled.
