## Usage

This chunk is a container for calendar.

``` php
<div id='[[+calendar_id]]'>
    [[+ec.Calendar]]
    <div class='cover'></div>
</div>
```

## Placeholders

| Name            | Description                          |
| --------------- | ------------------------------------ |
| Calendar        | Generated calendar with events.      |
| calendar\_id    | Unique id of calendar table on page. |
| class\_calendar | CSS classname for calendar table.    |
